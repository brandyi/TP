HOW TO ADD YOUR MEETING MEMORANDUM PDFs:

1. Place your PDF files in this folder (public/memorandum/)
2. Name them as: meeting-1.pdf, meeting-2.pdf, meeting-3.pdf, etc.
3. Update the meetings array in src/pages/Memorandum.tsx with:
   - The correct date
   - Meeting title
   - PDF filename (matching what you placed here)

Example PDFs to add:
- meeting-1.pdf
- meeting-2.pdf
- meeting-3.pdf

The website will automatically link to these files for viewing and downloading.
